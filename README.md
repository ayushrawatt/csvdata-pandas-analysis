# Data Analysis using Pandas

## Project Overview
This project was created as a learning exercise to practice data analysis using Python and Pandas.

The dataset contains ecommerce transaction data, and the goal of the project was to analyze the data and generate insights. The final output is a detailed Excel report created using Pandas.

## Tools Used
- Python
- Pandas
- Excel

## Output
The analysis generates an Excel file containing processed data and insights from the ecommerce dataset.
